# plantoid-led-master

### dependence
```
lua5.1

luarocks5.1 install luasocket
luarocks5.1 install luaposix
luarocks5.1 install lcurses
```

### runs serveur (console mode)
```
./led_master.lua
```
### replay dump
```
./led_master.lua replay dump/log.dump
```

### runs serveur with Love ( gui mode )
```
love .
```
